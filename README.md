#  Salon Wanita & Sewa Baju Adat Kartinian

Platform profil UMKM profesional yang dibina menggunakan **React** dan **Vite**. Website ini direka khas untuk mempamerkan perkhidmatan salon kecantikan serta katalog penyewaan baju adat tradisional dengan pengalaman pengguna yang lancar dan responsif.

##  Ciri-Ciri Utama

* **Arkitektur Multi-Halaman:** Menggunakan `react-router-dom` untuk navigasi yang lebih teratur antara Beranda, Katalog, Perkhidmatan Salon, dan Kontak.
* **Katalog Real-Work:** Galeri visual yang mempamerkan koleksi sebenar baju adat (Jawa, Kalimantan, Sulawesi, Sumatra) untuk meningkatkan kepercayaan pelanggan.
* **Panduan Ukuran (Size Guide):** Jadual rujukan saiz (S hingga XL) untuk memudahkan pelanggan memilih baju yang sesuai.
* **Integrasi WhatsApp Pintar:** Butang tempahan dinamik yang menjana pesanan teks automatik terus ke pemilik perkhidmatan.
* **Reka Bentuk Responsif & Mewah:** Menggunakan tipografi *Playfair Display* untuk kesan anggun dan *Inter* untuk kebolehbacaan yang moden.

##  Struktur Folder

Projek ini mengikuti struktur modular untuk memudahkan penyelenggaraan kod:

```text
src/
├── assets/         # Imej koleksi baju adat & gaya CSS global
├── components/     # Komponen UI (Navbar, Footer, ServiceCard, WhatsAppBtn)
├── data/           # Fail data statik untuk servis dan galeri
├── pages/          # Halaman utama (Home, Catalog, Salon, Contact)
├── App.jsx         # Konfigurasi routing utama
└── main.jsx        # Titik kemasukan aplikasi

```


##  Teknologi yang Digunakan

* **React 18** - Library UI utama.
* **Vite** - Alat binaan (build tool) yang pantas.
* **React Router Dom** - Pengurusan navigasi halaman.
* **FontAwesome** - Ikonografi untuk perkhidmatan dan sosial media.
* **Google Fonts** - *Playfair Display* & *Inter*.
* 

*Projek ini merupakan sebahagian daripada inisiatif digitalisasi UMKM untuk meningkatkan kualiti perkhidmatan dan capaian pelanggan.*
